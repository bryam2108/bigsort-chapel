BEAUTIFUL PARALLEL CODE

El corazón del algoritmo.
Simple. Poderoso.

Esta es la esencia del paralelismo en Chapel. Dos tareas, una fusión. Sin hilos, mutexes ni complejidad.

Recursión paralela con coforall
Cutoff inteligente para mejor localidad de caché
Zero-overhead abstractions
parallelMergeSort.chpl
// Recursión paralela elegante
coforall i in 0..1 {
  if i == 0 then
    parallelMergeSort(data, lo, mid);
  else
    parallelMergeSort(data, mid + 1, hi);
}

// Luego fusionamos las mitades ordenadas
merge(data, lo, mid, hi);
